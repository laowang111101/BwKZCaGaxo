# Python 基于模板的药品名称识别系统 python1128

## 项目概述

本项目是一个基于Python的药品名称识别系统，采用B/S架构，结合了MVC模式进行设计。系统主要利用自然语言处理技术，通过模板匹配的方式识别文本中的药品名称。

## 技术栈

- **架构**: B/S、MVC
- **系统环境**: Windows
- **开发环境**: VS Code、Python、Jupyter、MySQL
- **技术栈**: Python、MySQL、HTML、CSS、JavaScript、Flask

## 功能模块

### 系统角色

系统设计了两个主要角色：用户、管理员。

### 前台系统

- **首页**: 用户可以匿名访问首页，查看药品信息的简单介绍。
- **注册与登录**: 用户通过填写注册信息（用户名、密码、真实姓名等）进行注册，并登录使用系统。
- **个人中心**: 用户可以管理个人信息，如用户名、密码等。
- **系统信息查看**: 用户可以查看系统基本信息及药品数据更新情况。
- **文本提取**: 用户输入文本，系统自动识别出药品名称。
- **文本搜索**: 用户可通过关键词或条件进行药品信息检索。

### 后台管理系统

- **用户信息管理**: 管理员可以管理用户个人信息。
- **文本管理**: 管理员维护网站文本信息，包括提取记录的查看和修改。
- **管理员信息管理**: 管理员可以查看和修改自身信息。
- **用户信息管理**: 管理员负责用户信息的实时管理和权限控制。
